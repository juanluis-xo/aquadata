# AquaData - Plataforma Educativa ODS 6

Plataforma educativa digital para promover la gestión sostenible del agua.

## Instalación

1. Importar el archivo `database/schema.sql` en MySQL
2. Configurar las credenciales de base de datos en `config/database.php`
3. Colocar todos los archivos en el servidor web (Apache/Nginx con PHP)
4. Acceder a `index.html` desde el navegador

## Estructura del Proyecto

- `/api` - Endpoints PHP para el backend
- `/config` - Configuración de base de datos
- `/css` - Estilos CSS
- `/js` - JavaScript del frontend
- `/database` - Script SQL de la base de datos
